# CSS3 Neumorphic Social Media Icons | CodingNepal

Um projeto simples de **ícones de redes sociais com efeito neumorphism** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar **social media icons com estilo neumorphism**, utilizando sombras suaves para um visual moderno e elegante.

## 🛠️ Tecnologias

- HTML
- CSS (box-shadow, neumorphism)

## 🎯 Funcionalidades

- Ícones de redes sociais estilizados
- Efeito neumorphism
- Design moderno e minimalista

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
